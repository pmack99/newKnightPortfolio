Portfolio

ISS Tracker

The "International Space Station Tracker" gives the user the real time location and movement data of the ISS as it circles the Earth. In the ISS Pass section, the upcoming date, time and location of any upcoming future ISS pass above any city in the world can be predicted. The included NASA Image feed refreshes every 30 seconds giving a new ISS photo to review.

Crystal Game

This project was an exercise in generating random numbers. The game is simple, you need to click one of the four colored crystals. Each one of these crystals has a random number that will be subtracted from the "magic number". The goal is to get the magic number down to zero without going over.

Word Finder Game

This project was to create a basic "hangman" style word game. I themed it to MLB baseball team names. Blank spaces are generated on the screen equal to the length of the team name chosen randomly from an array. The Player guesses a letter, if the letter is present in the word, the blank spaces are replaced with the letter at the correct space.

Trivia Game

This project was to make a trvia style game utilizing JQuery and Javascript. Included features are a countdown timer and a scoreboard.

Evil Friend Finder

The application asks you to enter your name and image URL. 10 questions are then asked where you choose how strongly you agree or disagree (from 1 to 5) with the question. The answers are compared to the 10 "evil" friends stored in the friends.js file. The evil friend that is most compatable will be displayed in a modal. The technologies used in this exercise include Node.js, Express, and Path.
